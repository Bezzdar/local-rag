# Структура программы

## Термины

Ноутбук - локальная папка, архив, проектная папка. В ней пользователь объединяет набор документов для работы с ними. Создание нескольких ноутбуков необходимо для разграничения информации, структурирования и оптимизации обработки.

## Описание общих функций
1) Программа представляет собой локальный RAG-ассистент для работы с файлами на ПК пользователя. 
2) Подключение всех моделей (LLM, embedding, reranker) осуществляется с локального сервера предприятия. 
3) Программа предназначена для работы внутри закрытых сетей без доступа к сети интернет.

## Описание меню программы
### Главное меню

При входе в программу пользователь попадает на стартовое меню. В стартовом меню ему предлагается создать, выбрать, переименовать, удалить "ноутбук" (далее просто ноутбук).

При нажатии на кнопку ноутбука пользователь переходит в Основное рабочее окно.

Правое сворачиваемое вправо меню представляет собой меню настроек. 
В нем доступны сворачиваемые блоки:
1) Провайдер LLM - Блок настроек LLM, который состоит из нескольких  сворачиваемых подблоков.
	1) Настройки чата - настройки подключения LLM модели
	2) Эмбеддинг - настроки эмбеддинга
	3) Далее будут появляться новые подблоки
2) Глобальные настройки парсинга - Блок настроек парсинга и чанкинга
В дальнейшем блоки и подблоки будут расширяться

Все настройки сохраняются. По умолчанию, при пуске LLM не подключено (Режим None) и пользователь нужно самостоятельно запускать провайдера LLM по последним сохраненным настройкам или изменяя их.
### Основное рабочее окно
Основное рабочее окно состоит из трех секций:
1) Центральная секция
2) Левая сворачиваемая секция
3) Правая сворачиваемая секция
