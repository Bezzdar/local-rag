# RAG

Тестовый код для RAG‑ассистента.

## Структура проекта (быстрая карта)

### Core (основной контур)
- `streamlit_app.py` — основной UI и основной поисковый pipeline.
- `app/` — бизнес‑логика (индексация, поиск, LLM‑вызовы, настройки).
- `parsers/text_extraction.py` — извлечение контента из документов.

### Tools (вспомогательные утилиты)
- `scripts/index_all_folders.py` — пакетный запуск индексации всех папок.
- `scripts/trigger_indexing.py` — индексация одной папки через CLI.
- `pdf_chunk_tester.py` — локальная диагностика качества PDF‑чанкинга.
- `app/chunk_editor.py` — отдельный Streamlit‑редактор чанков (вне основного UI).

### Runtime/Artifacts
- `data/docs/` — входные документы.
- `data/index/` — локальные индексные файлы.
- `llama.cpp/` — локальные бинарники и инструменты запуска моделей.

## Текущие шаги оптимизации структуры

- Добавлен `.gitignore` для исключения кэшей, venv и runtime‑артефактов индекса.
- Убран неиспользуемый код в `streamlit_app.py` (лишние импорты и неиспользуемые launch‑обёртки).
- Утилиты пока сохранены без удаления; финальное решение по матрице файлов согласуется отдельно.
